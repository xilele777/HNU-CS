# 简答题（10 分）

在某机器中，int 型、unsigned int 型均占一个字节（8bit），short 型占用 4 个 bit，并且有符号数均采用补码表示。float 型也用一个字节来表示，并遵循 IEEE754 规范：最高位表示符号，中间 4 位为阶码，最后 3 位为尾数。

## （1）（4 分）请以此机器为例，用数字 225 来解释浮点数的"向偶数舍入"和"int 转换为 float 时，数字不会溢出，但可能被舍入"。

## （2）（6 分）假设有如下代码:

```c
float f = -1;
int *pi = &f;
int i = *pi;
short s = (short)i;
unsigned int ui = (unsigned int)s;
float *pf = &ui;
```

请写出 f 的二进制表示，ui 的二进制表示，*pf 的值。
