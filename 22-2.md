# 题目解析

## 答案汇总

| 空号 | 答案 |
|------|------|
| **(1)** | `23` |
| **(2)** | `6` |
| **(3)** | `(j > 15) ? 0 : (j - 15)` |
| **(4)** | `array1[i][j] - 2 * array2[k]` |
| **(5)** | `sum + array2[0] + array1[0][0]` |

---

## 逐空解析

### (1) Y = 23

关键证据在汇编第 60-62 行：

```asm
imull   $23, %eax, %eax       ; eax = i * 23
addl    -16(%ebp), %eax       ; eax = i * 23 + j
movl    array1(,%eax,4), %ecx ; 访问 array1[i * 23 + j]
```

二维数组 `array1[i][j]` 按行优先存储，地址为 `array1 + (i × Y + j) × 4`。这里乘以 23，说明 **Y = 23**。

另外循环条件 `cmpl $22, -16(%ebp)` 检查 `j ≤ 22`，即 `j < 23`，与 `j < Y` 一致。

> **补充知识：为什么 `array1[i][j]` = `array1[i * 23 + j]`？**
>
> C 语言中二维数组采用**行优先存储（Row-major order）**。`int array1[16][23]` 在内存中是一块连续的一维空间，按行依次排列：
>
> ```
> 内存地址:  低 ──────────────────────────────────────▶ 高
>            ┌──────────┬──────────┬─────┬──────────────┐
>            │  第 0 行  │  第 1 行  │ ... │   第 15 行   │
>            │ 23 个元素 │ 23 个元素 │     │  23 个元素   │
>            └──────────┴──────────┴─────┴──────────────┘
> ```
>
> 要访问第 `i` 行、第 `j` 列的元素，需要**跳过前面 i 个完整行**：
>
> ```
> 偏移量 = i × 23 + j
> ```
>
> 在汇编层面，`array1` 只是一个首地址，编译器将 `array1[i][j]` 翻译为 `*(array1 + i × 列数 + j)`。

### (2) sum = 6

汇编第 39 行直接赋值：

```asm
movl    $6, -12(%ebp)         ; sum = 6
```

结合栈布局，`-12(%ebp)` 对应变量 `sum`，初始值为 **6**。

---

### (3) k = (j > 15) ? 0 : (j - 15)

汇编第 48-56 行：

```asm
cmpl    $15, -16(%ebp)        ; 比较 j 和 15
jg      .L3                   ; 若 j > 15，跳转到 L3
movl    -16(%ebp), %eax       ; eax = j
subl    $15, %eax             ; eax = j - 15
jmp     .L4
.L3:
movl    $0, %eax              ; eax = 0
.L4:
movl    %eax, -8(%ebp)        ; k = eax
```

逻辑：若 `j > 15`，`k = 0`；否则 `k = j - 15`。写成 C 的三目运算符即 **`(j > 15) ? 0 : (j - 15)`**。

---

### (4) array1[i][j] - 2 * array2[k]

汇编第 57-71 行逐步计算 `array2[i] += ...`：

```asm
movl    -20(%ebp), %eax         ; eax = i
movl    array2(,%eax,4), %edx   ; edx = array2[i]           ← 读取旧值

movl    -20(%ebp), %eax         ; eax = i
imull   $23, %eax, %eax         ; eax = i * 23
addl    -16(%ebp), %eax         ; eax = i * 23 + j
movl    array1(,%eax,4), %ecx   ; ecx = array1[i][j]        ← 取 array1[i][j]

movl    -8(%ebp), %eax          ; eax = k
movl    array2(,%eax,4), %ebx   ; ebx = array2[k]           ← 取 array2[k]

movl    $0, %eax                ; eax = 0
subl    %ebx, %eax              ; eax = -array2[k]
addl    %eax, %eax              ; eax = -2 * array2[k]      ← 乘以 -2

addl    %ecx, %eax              ; eax = array1[i][j] - 2*array2[k]
addl    %eax, %edx              ; edx = array2[i] + 上式      ← 累加

movl    -20(%ebp), %eax         ; eax = i
movl    %edx, array2(,%eax,4)   ; array2[i] = 新值           ← 写回
```

数据流图：

```
                     ┌──────────────┐
                     │  array2[i]   │ ──────────────┐
                     └──────────────┘               │
                                                    ▼
┌──────────┐    ┌──────────────┐              ┌──────────┐
│ i × 23+j │──▶│ array1[i][j] │──────────────▶│   ADD    │──▶ array2[i]
└──────────┘    └──────────────┘              └──────────┘
                                                    ▲
┌──────────┐    ┌──────────────┐                    │
│    k     │──▶│  array2[k]   │──▶ ×(-2) ──────────┘
└──────────┘    └──────────────┘
```

所以累加的值为 **`array1[i][j] - 2 * array2[k]`**。

> **关于 `movl array2(,%eax,4), %edx` 的寻址方式：**
>
> AT&T 语法中，内存寻址格式为：`位移量(基址寄存器, 变址寄存器, 比例因子)`
>
> 有效地址 = 位移量 + 基址 + 变址 × 比例因子
>
> | 部分 | 值 | 含义 |
> |------|-----|------|
> | 位移量 | `array2` | 数组起始地址 |
> | 基址寄存器 | 空 | 无 |
> | 变址寄存器 | `%eax` | 索引 i |
> | 比例因子 | `4` | sizeof(int) = 4 |
>
> 即：`edx = Mem[&array2 + i × 4] = array2[i]`

### (5) sum + array2[0] + array1[0][0]

汇编第 80-88 行（`.L5` 标号处，循环结束后）：

```asm
.L5:
movl    array2, %eax          ; eax = array2[0]
movl    %eax, %edx            ; edx = array2[0]
addl    -12(%ebp), %edx       ; edx = array2[0] + sum
movl    array1, %eax          ; eax = array1[0] = array1[0][0]
addl    %edx, %eax            ; eax = sum + array2[0] + array1[0][0]
; ret 返回，eax 作为返回值
```

返回值为 **`sum + array2[0] + array1[0][0]`**。

## 栈变量布局总结

| 栈偏移 | 变量 | 说明 |
|--------|------|------|
| `-20(%ebp)` | `i` | 循环变量，初值 1，每次 +1 |
| `-16(%ebp)` | `j` | 循环变量，初值 2，每次 +1 |
| `-12(%ebp)` | `sum` | 累加和，初值 6 |
| `-8(%ebp)` | `k` | 由 j 计算得到 |

## 完整程序还原

```c
#include <stdio.h>
#define X 16
#define Y 23
int array1[X][Y];
int array2[X];
int test()
{
    int sum = 6;
    int i = 0, j = 0, k = 0;
    for (i = 1, j = 2; i < X && j < Y; i++)
    {
        k = (j > 15) ? 0 : (j - 15);
        array2[i] += array1[i][j] - 2 * array2[k];
        j++;
    }
    return sum + array2[0] + array1[0][0];
}
int main()
{
    return 0;
}
```
