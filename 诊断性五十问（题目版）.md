# 2026 春"计算机系统"诊断性五十问

以下五十个问题，旨在帮助同学定位自己在本门课程的学习过程中到底在哪里"掉链子"了，属于诊断性提问式复习引导，与期末考题不存在任何直接对应关系。

强烈建议诸君（不要借助AI）独立作答，真实锚定属于自己的重难点。答不出来的部分就是需要重点复习的章节。

---

## 数据的机器级表示 (1-10)

1. `int x = -1`，在32位机器上，x的十六进制表示是什么？用补码解释为什么。

2. `unsigned int a = 0; a-1` 的结果是多少？这说明了无符号数的什么性质？

3. 为什么 C语言中 `(int)(-1)>(unsigned int)(0)` 的结果是True？底层发生了什么？

4. float 只有32位，却能表示比 int 大得多的数，代价是什么？举一个具体的例子说明精度损失。

5. IEEE 754中，+0和-0的位模式不同，但+0==-0为true。NaN != NaN 为什么是 true？

6. 一个int 数组a[4]存在内存里，a[0]的地址是0x100。a[2]的地址是多少？`*(a+3)`和 `a[3]`有什么区别？

7. 大端序和小端序的区别是什么？int x=0x01020304 在小端机器上，从低地址到高地址，4个字节依次是什么？

8. 左移 `x<<k` 等价于乘以 $2^k$，这个说法在什么情况下会失效？

9. 为什么 C语言中比较两个 float 是否相等通常用 `fabs(a-b)<eps` 而不是 `a==b`？

10. `sizeof(int)` 在32位和64位系统上分别是多少？`sizeof(int*)` 呢？为什么指针大小和平台相关？

## 程序的机器级运行 (11-20)

    ```c

    if (a > b) {...}
    ```

16. 递归函数在汇编层面是如何实现的？和循环相比，它的性能开销来自哪里？

17. 以下两段代码在汇编层面有什么不同？编译器会如何处理 switch？

    ```c
    if (x == 1)... else if(x == 2)... else if(x == 3)...
    switch(x) { case 1:... case 2:... case 3:... }
    ```

18. 函数指针在底层实质是什么？

19. 为什么局部变量在函数返回后就不能访问了？返回一个局部变量的指针会发生什么？

20. （跨章节）一个局部 int 数组 `buf[8]` 存在栈上，它的元素在内存中是连续的。如果发生缓冲区溢出，结合虚存和栈的布局，说明为什么有时候溢出会 segfault，有时候却"悄悄"成功？

## 存储层次与局部性 (27-30)

27. 什么是时间局部性和空间局部性？以下哪段代码局部性更好？为什么？

    ```c
    // A:按行遍历
    for(i=0;i<N;i++) for(j=0;j<N;j++) sum += a[i][j];
    // B:按列遍历
    for(j=0;j<N;j++) for(i=0;i<N;i++) sum += a[i][j];
    ```

28. 存储层次金字塔大致是什么样的？为什么需要多个存储设备，且它们能在一个计算机系统里协调工作？

29. 矩阵乘法 $C=A\times B$，朴素的三重循环 i-j-k 顺序和 i-k-j 顺序，哪个（更尊重 Cache）性能更好？请从局部性角度分析。

30. （跨章节）加载实际上完成了什么工作？加载之前必须要生成新进程，为什么？
